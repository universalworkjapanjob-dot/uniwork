本体ダウンロード

アプリ本体ファイル一式・データのダウンロードができます。別アカウント・別端末への移行に使用してください。

アプリ全体をZIPでダウンロード

index.html / css / js ファイル一式をZIP形式でダウンロードします。

現在のデータをJSONでダウンロード

利用者・実績・受給者証などの全データをJSONとして保存します。別端末で「設定→データ管理→復元」から読み込めます。

アプリ＋データを一括ダウンロード

アプリ本体ZIPと現在のデータJSONを両方まとめてダウンロードします。

別端末・別アカウントで使い始める手順

  1. 「ZIPダウンロード」でアプリ本体を取得
  2. 「データJSON出力」で現在のデータを保存
  3. 別端末でZIPを解凍し index.html をブラウザで開く
  4. 設定 → データ管理 →「JSONファイルを選択して復元」でデータを読み込む
  5. Google連携する場合は 設定 → データ管理 → Google連携設定 を入力して保存

データ管理

全データをJSONファイルとして書き出し・読み込みができます。別アカウントへの移行や定期バックアップに使用してください。

バックアップ（書き出し）

現在のデータ全件をJSONファイルとしてダウンロードします。

復元（読み込み）

バックアップしたJSONファイルを読み込んでデータを復元します。
※ 現在のデータは上書きされます。
