# UNIWORK 就労継続支援B型 利用実績管理システム

## 概要
就労継続支援B型事業所向けの利用実績管理Webアプリ。
ブラウザだけで動作するスタンドアロン版（LocalStorage保存）。

## ログイン情報
- 管理者: `admin` / `admin123`
- 職員: `staff` / `staff123`

## 主な機能
- ダッシュボード（KPI・グラフ）
- 出勤予定表（月次・セル編集・一括設定・途中変更・キャンセル）
- 日別実績入力（通所/在宅/キャンセル/工賃自動計算）
- 月次実績一覧（CSV出力）
- 利用者管理（個別工賃設定）
- 受給者証管理（期限アラート）
- モニタリング管理
- 工賃管理（グラフ・CSV）
- 分析（6タブ・各種グラフ）
- 月次締め・請求チェック
- 情報共有ボード
- 帳票出力（CSV）
- 設定・データ管理

## データ管理（バックアップ・復元）
設定 → データ管理 にて以下が可能：
- **全データをJSONでダウンロード**（バックアップ）
- **JSONファイルを選択して復元**（別環境への移行）

## ファイル構成
- `index.html` ：メインHTML
- `css/style.css`：スタイル
- `js/data.js`：サンプルデータ・LocalStorage管理
- `js/app.js`：全機能ロジック

## 工賃ルール
- 通所: 850円/時（個別設定可）
- 在宅: 550円/時（個別設定可）
- 30分単位で自動減額
- 最大5時間（延長16:00まで）

## 事業所設定
- 午前 10:00〜12:00
- 午後 13:00〜15:00
- 延長 〜16:00（最大5時間）
- 利用形態：通所 / 在宅 / 農場

## Google連携（本番化）
設定 → データ管理 → Google連携設定 にて以下を入力：
1. OAuth Client ID
2. Apps Script Web App URL
3. Spreadsheet ID
4. Drive フォルダ ID
